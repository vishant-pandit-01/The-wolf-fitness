# 🐺the Wolf Fitness - Gym Website

A modern, responsive and fully animated gym website built with **Django, HTML, CSS, and JavaScript**.

---

## 🚀 About Project

**Wolf Fitness** is a fitness website designed to provide a premium gym experience online.  
It includes workout plans, membership options, trainer profiles, and contact system.

---

## ✨ Features

- ⚡ Responsive modern UI design
- 🐺 Animated navbar with logo
- ⌨️ Typing text animation
- 💪 Workout plans section
- 🏋️ Membership packages
- 📊 Performance dashboard
- 📩 Contact form (Formspree integration)
- 🧾 Register form with validation
- 🖼️ Static image handling (Django)
- 🔥 Smooth scrolling and animations

---

## 🛠️ Tech Stack

- Django (Backend)
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
